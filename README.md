# 🏦 Loan Approval Prediction System

## 📌 Overview
This is a Machine Learning web application that predicts whether a loan application will be approved or rejected based on applicant financial and personal details.

The project uses:
- FastAPI (Backend API)
- Streamlit (Frontend UI)
- Scikit-learn ML model

---

## 🚀 Features
- Predict loan approval in real-time
- User-friendly UI using Streamlit
- Fast API response using FastAPI
- Feature engineering (asset ratio, income ratio, etc.)

---

## 🛠 Tech Stack
- Python
- FastAPI
- Streamlit
- Pandas, NumPy
- Scikit-learn
- Joblib

---
